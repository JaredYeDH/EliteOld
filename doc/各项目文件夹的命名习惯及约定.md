各项目文件夹的命名习惯及约定
==========================

无论是子项目还是主项目，各文件夹都有几乎一致的命名方式和含义，下面简要阐述：

| 目录      | 功能简介                              |
| ---     | ---                               |
| src     | 源文件目录，用来存放‘cpp’源码和不需要公开接口的‘.h’头文件 |
| include | 公开类的头文件和接口文件                      |
| doc     | 文档目录，一般是手写文档和文档图片资源的存放            |
| conf    | 配置文件                              |
| lib     | 依赖的库文件，既有自己的组件，也有其他项目库            |
| scripts | 脚本文件                              |
| tools   | 工具链目录                             |
| test    | 单元测试文件目录                          |
